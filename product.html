Multiple detection methods

AIVerify combines visual AI analysis, EXIF metadata inspection, and error-level analysis (ELA) into a single combined score.

Each method contributes independently so you can see which factor carried the most weight.

Keyboard Shortcuts
Ctrl + Shift + A Capture a screen region and analyze as image
Ctrl + Shift + V Record a screen region for 5 seconds and analyze as video
Ctrl + Shift + T Open text input box to paste or type text
Ctrl + Shift + X Analyze text currently copied to clipboard

All hotkeys can be customized in the app settings.
